# 🏫 School Management System

A comprehensive school management system built with Laravel 13.4.0 and Filament PHP Admin Panel v5.5.0.

## 📋 Overview

This School Management System provides a complete administrative interface for managing schools, students, academic years, and other educational resources. Built with Laravel's elegant framework and Filament's powerful admin panel, it offers a modern, responsive, and user-friendly interface for school administrators.

## ✨ Features

### Current Features
- 🏢 **School Management**
  - Create, read, update, and delete schools
  - Manage school details (name, code, contact info, address)
  - Upload school logos
  - Activate/deactivate schools

- 👨‍🎓 **Student Management**
  - Complete student profiles
  - Track admission numbers
  - Manage parent/guardian information
  - Record medical notes and allergies
  - Track enrollment dates
  - Filter and search students

- 📚 **Academic Management** (Coming Soon)
  - Academic year management
  - Class/Form management
  - Subject management
  - Timetable management

- 👨‍🏫 **Staff Management** (Coming Soon)
  - Teacher profiles
  - Staff attendance
  - Leave management

## 🚀 Technology Stack

- **Backend Framework**: Laravel 13.4.0
- **Admin Panel**: Filament PHP v5.5.0
- **PHP Version**: PHP 8.5.1+
- **Database**: SQLite (configurable for MySQL/PostgreSQL)
- **Frontend**: Livewire v4.2.4, Tailwind CSS
- **Authentication**: Laravel Breeze/Sanctum

## 📦 Installation

### Prerequisites
- PHP >= 8.5.1
- Composer
